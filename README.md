This is the MVP for a open data project that will display fuel economy data from the US Department of Energy. 

You can find the data set at fueleconomy.gov.

###Services:
* Year
* Make
* All Car Data
* YourSaveSpend
* HighwayMPG
* CityMPG

###Node Installation:

####Install application
```
npm install
```

####Install Bower
```
npm install [-g] bower
```


###Django Installation:

#### Django dependancies

```
pip install -r requirement_file.txt
```
